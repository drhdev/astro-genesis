# AstroWind Reference Notes

AstroWind influenced the template shape, not the implementation details.

Borrowed principles:

- Content-rich home page.
- Reusable widget components.
- Blog, RSS, sitemap, and SEO-first defaults.
- Dark mode support.
- Simple deployment story.
- Minimal JavaScript unless interactivity is needed.

Updated choices:

- Astro 6 instead of Astro 5.
- Tailwind 4 Vite plugin instead of deprecated Astro Tailwind integration.
- Server output with explicit prerendering to demonstrate static and dynamic routes.
- Server Islands, Actions, Sessions, Live Content Collections, Fonts API, and CSP.
- Docker-first Node standalone deployment.
